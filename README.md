# FloraVision

A responsive plant e-commerce landing page built using React.js and Tailwind CSS.

## Tech Stack

- React.js
- Tailwind CSS
- Vite

## Installation

```bash
git clone https://github.com/Aazimkhursheed/FloraVision.git
cd FloraVision
npm install
npm run dev
```

## Build

```bash
npm run build
```
